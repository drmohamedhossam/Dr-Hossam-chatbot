# 🧠 د. محمد حسام - Chatbot

## خطوات الرفع على Vercel

### 1️⃣ احصل على Gemini API Key مجاناً
- افتح: https://aistudio.google.com/app/apikey
- اضغط "Create API Key"
- انسخ المفتاح

### 2️⃣ ارفع المشروع على GitHub
- افتح: https://github.com/new
- اعمل repo جديد اسمه: dr-hossam-chatbot
- ارفع كل الملفات

### 3️⃣ ارفع على Vercel
- افتح: https://vercel.com
- اضغط "Add New Project"
- اختار الـ repo من GitHub
- في "Environment Variables" حط:
  - Name: VITE_GEMINI_API_KEY
  - Value: (المفتاح اللي نسخته)
- اضغط Deploy!

### 🎉 خلاص!
هتاخد لينك زي: https://dr-hossam-chatbot.vercel.app
